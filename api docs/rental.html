Rental API v1.0 - Documentation

trip creation

Description
Inserts a trip record into the database and returns a JSON object containing the trip's details.

Input 
{
    "DriverID": 1,
    "PlateNo": "SMU1234A",
    "StartLocation": 1,
    "BookingDuration": 2.5,
    "TotalFare": 15.25
}

HTTP request
POST http://127.0.0.1:5001/rental

vehicle update

Description
Change the vehicle given the rental ID.

Input 
{
    "PlateNo": "SMU1234A"
}

HTTP request
PUT http://127.0.0.1:5001/rental/1
where the last parameter value is an integer which represents the rental ID.

end trip

Description
Ends the trip given the rental ID.

Input 
{
    "EndLocation": 1
}

HTTP request
PUT http://127.0.0.1:5001/rental/endtrip/1
where the last parameter value is an integer which represents the trip ID.
