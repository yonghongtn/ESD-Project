Vehicle API v1.0 - Documentation

get all vehicles

Description
Returns a collection of vehicles in JSON format.

HTTP request
GET http://127.0.0.1:5000/rentalvehicle

get vehicles by brand

Description
Returns a collection of vehicles matching the brand keyword in JSON format.

HTTP request
GET http://127.0.0.1:5000/rentalvehicle/SGA1234B
where the last parameter is 8 character long string which represents the vehicle plate number.
